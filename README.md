# container-images

Self-hosted container images that I use.

This repo builds hardened images on Red Hat Hummingbird distroless bases. The goal is to replace upstream images with images built with pinned checksums, signed GHCR publishes, SBOMs, and provenance attestations.

Images will be published to:

- `ghcr.io/major/container_name`

Pull requests are welcomed! 👋
